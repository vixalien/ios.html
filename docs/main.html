iOS.html

iOS.html is a reproduction of iOS Human Design Interface in HTML, so that we can use it in our webapps.

We all like the appearance of iOS, no, so this is what I tried: I made an HTML project, call it a boilerplate, so that we can design webapps with the appearance of iOS

Details

Maintained by: vixalien only

Github: vixalien/ios.html

Web: vixalien.github.io/ios.html/docs

Resources

You might want to check-out iOS Human Design Interface, to see how the various components appear vs iOS.html or see missing components.
